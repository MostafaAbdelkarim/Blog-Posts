## Basic Overview

Implementing CRUD operations using Ruby On Rails and Bootstrap

## How to use

- Clone the repo.
- Open CMD
- cd --project-path
- 'rails s' or 'rails server'
- You can start experimenting using this link:

```text
http://localhost:3000/
```
